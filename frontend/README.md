Prism Agro
Prism Agro is a secure, decentralized agricultural marketplace designed to eliminate middlemen and reduce post-harvest loss in the Nigerian supply chain. By connecting Farmers, Buyers, and Logistics providers directly, we ensure fair pricing and produce freshness.

Core Value Proposition
Verified Trust: Mandatory identity verification via NIN-API for all stakeholders.

Freshness Tracking: A proprietary time-series algorithm that monitors produce quality from harvest to delivery.

Optimized Logistics: Integrated fulfillment tracking to bridge the gap between rural farms and urban markets.

Technical Stack
Frontend: Next.js 16.2.1 (App Router), Tailwind CSS

Backend: Node.js, Express

Database: MongoDB

Fintech: Interswitch / Quickteller API

Getting Started
Bash
# Install dependencies
npm install

# Run development server
npm run dev
Open http://localhost:3000 to view the platform.
